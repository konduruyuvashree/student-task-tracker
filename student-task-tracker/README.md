# 📚 StudyTrack — Student Task Tracker

A web-based task management application built for college students to track assignments, deadlines, and submission status across multiple subjects.
Project built with Flask and SQLite

---

## 🎯 Problem Statement

College students struggle to track assignments, deadlines, and submission status across multiple subjects. StudyTrack solves this by providing a clean, fast, and intuitive web app to manage all academic tasks in one place.

---

## ✨ Features

- ➕ **Add / Edit / Delete Tasks** — Full CRUD functionality
- 📅 **Deadline Reminders** — Auto-alerts for overdue and upcoming tasks
- 🔍 **Subject-wise Filtering** — Filter tasks by subject or status
- 🌙 **Dark Mode** — Toggle between light and dark themes
- 🐍 **Python Flask Backend** — REST API with SQLite database
- 📊 **Live Stats Dashboard** — Total, Pending, Completed, Due Today,Task Done

## ⚙️ Setup Instructions
Use Git Bash

### 1. Clone the Repository
```bash
git clone https://github.com/YOUR_USERNAME/student-task-tracker.git
cd student-task-tracker
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the App
```bash
python app.py
```

### 4. Open in Browser
```
http://127.0.0.1:5000
```
